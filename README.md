# Power BI Sales Dashboard

## Project Overview
This project demonstrates an interactive sales dashboard created in Power BI. It provides visual insights into sales performance across products, regions, and time periods.

## Files
- `sales_dashboard.pbix` : Power BI file with all visuals and data model.
- `dashboard_overview.png` : Screenshot of the main dashboard.
- `sales_by_region.png` : Screenshot highlighting regional sales performance.

## Features
- Interactive charts and slicers for filtering data.
- Trend analysis by time period.
- Region-wise and product-wise performance metrics.

## Tools Used
- Microsoft Power BI
